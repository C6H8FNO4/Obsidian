元音是气流不受阻碍发出的声音。分为：
单元音（12个）
双元音（8个）
# 单元音
### 🗣️ 英语单元音（12个）

| 分类      | 音标   | 例词    | 发音说明           |
| :------ | :--- | :---- | :------------- |
| **前元音** | /iː/ | see   | 长音，嘴角拉开，舌位高前   |
|         | /ɪ/  | sit   | 短音，舌稍低于 /iː/   |
|         | /e/  | pen   | 中前短音           |
|         | /æ/  | cat   | 前低元音，张口较大      |
| **中元音** | /ʌ/  | cup   | 中央偏低，短音        |
|         | /ɜː/ | bird  | 中央偏中，长音（英音无卷舌） |
|         | /ə/  | about | 弱读音，常用于非重读音节   |
| **后元音** | /uː/ | too   | 长音，双唇圆         |
|         | /ʊ/  | book  | 短音，唇圆但较松弛      |
|         | /ɔː/ | saw   | 长音，舌位中后        |
|         | /ɒ/  | hot   | 短音（英音常见）       |
|         | /ɑː/ | car   | 长音，舌位低后，口型大    |
# 双元音
### 🌈 英语双元音（8个）

| 音标 | 例词 | 发音变化趋势 | 发音说明 |
|:--|:--|:--|:--|
| /eɪ/ | name | e → ɪ | 从中前元音滑向高前元音 |
| /aɪ/ | my | a → ɪ | 从低元音滑向高前元音 |
| /ɔɪ/ | boy | ɔ → ɪ | 从中后圆唇音滑向高前元音 |
| /aʊ/ | now | a → ʊ | 从低元音滑向高后圆唇音 |
| /əʊ/ | go | ə → ʊ | 从中元音滑向高后圆唇音 |
| /ɪə/ | ear | ɪ → ə | 从高前元音滑向中元音 |
| /eə/ | hair | e → ə | 从中前元音滑向中元音 |
| /ʊə/ | tour | ʊ → ə | 从高后元音滑向中元音 |
