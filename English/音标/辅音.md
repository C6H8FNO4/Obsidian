# 🔊 英语辅音分类：清辅音 & 浊辅音

## 🧊 清辅音（Voiceless Consonants）
> 发音时声带**不振动**，气流受阻产生摩擦或爆破声。

| 音标 | 类型 | 例词 | 发音说明 |
|:--|:--|:--|:--|
| /p/ | 爆破音 | pen | 双唇闭合，突然放开气流 |
| /t/ | 爆破音 | ten | 舌尖抵上齿龈，爆破出气 |
| /k/ | 爆破音 | cat | 舌后抵软腭，爆破出气 |
| /f/ | 摩擦音 | fan | 上齿轻咬下唇出气 |
| /θ/ | 摩擦音 | thin | 舌尖伸出上下齿之间 |
| /s/ | 摩擦音 | see | 舌尖靠近上齿龈，气流通过 |
| /ʃ/ | 摩擦音 | she | 舌面靠近硬腭，气流摩擦 |
| /tʃ/ | 破擦音 | church | 先阻后摩，类似“吃”音 |
| /h/ | 摩擦音 | hat | 喉部送气，声带不振动 |

➡️ 共 **9 个清辅音**

## 🔥 浊辅音（Voiced Consonants）
> 发音时声带**振动**，可感觉到喉咙震动。

| 音标 | 类型 | 例词 | 发音说明 |
|:--|:--|:--|:--|
| /b/ | 爆破音 | bed | 与 /p/ 相对，双唇闭合后振动声带放气 |
| /d/ | 爆破音 | dog | 舌尖抵齿龈后放气 |
| /g/ | 爆破音 | go | 舌后抵软腭后放气 |
| /v/ | 摩擦音 | van | 与 /f/ 相对，声带振动 |
| /ð/ | 摩擦音 | this | 与 /θ/ 相对，舌尖振动声带 |
| /z/ | 摩擦音 | zoo | 与 /s/ 相对，声带振动 |
| /ʒ/ | 摩擦音 | measure | 与 /ʃ/ 相对，舌面靠近硬腭 |
| /dʒ/ | 破擦音 | judge | 与 /tʃ/ 相对，声带振动 |
| /m/ | 鼻音 | man | 双唇闭合，气流经鼻腔 |
| /n/ | 鼻音 | name | 舌尖抵齿龈，气流经鼻腔 |
| /ŋ/ | 鼻音 | sing | 舌后抵软腭，气流经鼻腔 |
| /l/ | 边音 | leg | 气流从舌两侧流出 |
| /r/ | 近音 | red | 舌尖轻微卷起，不接触上腭 |
| /j/ | 半元音 | yes | 类似“耶”音 |
| /w/ | 半元音 | we | 双唇圆起，接近 /u/ 音 |
