# 🎧 英语48个国际音标总表（IPA for English）

> 版本：英式发音（Received Pronunciation, RP）  
> 共计：48 个音标 = 20 个元音 + 28 个辅音  

---

## 🗣️ 一、元音（Vowels）

### ① 单元音（12个）

| 分类 | 音标 | 例词 | 发音说明 |
|:--|:--|:--|:--|
| **前元音** | /iː/ | see | 长音，嘴角拉开，舌位高前 |
|  | /ɪ/ | sit | 短音，舌稍低于 /iː/ |
|  | /e/ | pen | 中前短音 |
|  | /æ/ | cat | 前低元音，张口较大 |
| **中元音** | /ʌ/ | cup | 中央偏低，短音 |
|  | /ɜː/ | bird | 中央偏中，长音（英音无卷舌） |
|  | /ə/ | about | 弱读音，常用于非重读音节 |
| **后元音** | /uː/ | too | 长音，双唇圆 |
|  | /ʊ/ | book | 短音，唇圆但较松弛 |
|  | /ɔː/ | saw | 长音，舌位中后 |
|  | /ɒ/ | hot | 短音（英音常见） |
|  | /ɑː/ | car | 长音，舌位低后，口型大 |

---

### ② 双元音（8个）

| 音标 | 例词 | 发音变化趋势 | 发音说明 |
|:--|:--|:--|:--|
| /eɪ/ | name | e → ɪ | 从中前元音滑向高前元音 |
| /aɪ/ | my | a → ɪ | 从低元音滑向高前元音 |
| /ɔɪ/ | boy | ɔ → ɪ | 从中后圆唇音滑向高前元音 |
| /aʊ/ | now | a → ʊ | 从低元音滑向高后圆唇音 |
| /əʊ/ | go | ə → ʊ | 从中元音滑向高后圆唇音 |
| /ɪə/ | ear | ɪ → ə | 从高前元音滑向中元音 |
| /eə/ | hair | e → ə | 从中前元音滑向中元音 |
| /ʊə/ | tour | ʊ → ə | 从高后元音滑向中元音 |

## 🔊 二、辅音（Consonants）

辅音在发音时气流受到阻碍。  
分为：**清辅音（voiceless）** 与 **浊辅音（voiced）**。

---

### 🧊 清辅音（9个）

> 声带不振动，气流摩擦或爆破产生声音。

| 音标 | 类型 | 例词 | 发音说明 |
|:--|:--|:--|:--|
| /p/ | 爆破音 | pen | 双唇闭合，突然放开气流 |
| /t/ | 爆破音 | ten | 舌尖抵上齿龈，爆破出气 |
| /k/ | 爆破音 | cat | 舌后抵软腭，爆破出气 |
| /f/ | 摩擦音 | fan | 上齿轻咬下唇出气 |
| /θ/ | 摩擦音 | thin | 舌尖伸出上下齿之间 |
| /s/ | 摩擦音 | see | 舌尖靠近上齿龈 |
| /ʃ/ | 摩擦音 | she | 舌面靠近硬腭，气流摩擦 |
| /tʃ/ | 破擦音 | church | 先阻后摩，类似“吃”音 |
| /h/ | 摩擦音 | hat | 喉部送气，声带不振动 |

---

### 🔥 浊辅音（15个）

> 声带振动，可感觉到喉咙震动。

| 音标 | 类型 | 例词 | 发音说明 |
|:--|:--|:--|:--|
| /b/ | 爆破音 | bed | 与 /p/ 相对，双唇闭合后振动声带放气 |
| /d/ | 爆破音 | dog | 舌尖抵齿龈后放气 |
| /g/ | 爆破音 | go | 舌后抵软腭后放气 |
| /v/ | 摩擦音 | van | 与 /f/ 相对，声带振动 |
| /ð/ | 摩擦音 | this | 与 /θ/ 相对，舌尖振动声带 |
| /z/ | 摩擦音 | zoo | 与 /s/ 相对，声带振动 |
| /ʒ/ | 摩擦音 | measure | 与 /ʃ/ 相对，舌面靠近硬腭 |
| /dʒ/ | 破擦音 | judge | 与 /tʃ/ 相对，声带振动 |
| /m/ | 鼻音 | man | 双唇闭合，气流经鼻腔 |
| /n/ | 鼻音 | name | 舌尖抵齿龈，气流经鼻腔 |
| /ŋ/ | 鼻音 | sing | 舌后抵软腭，气流经鼻腔 |
| /l/ | 边音 | leg | 气流从舌两侧流出 |
| /r/ | 近音 | red | 舌尖轻微卷起，不接触上腭 |
| /j/ | 半元音 | yes | 类似“耶”音 |
| /w/ | 半元音 | we | 双唇圆起，接近 /u/ 音 |

## ⚖️ 三、清浊辅音配对表

> 以下为常见的**清辅音（Voiceless）**与**浊辅音（Voiced）**对应关系。  
> 左侧为清音（声带不振动），右侧为浊音（声带振动）。

| 清辅音 | 浊辅音 | 类型 | 清音例词 | 浊音例词 | 说明 |
|:--|:--|:--|:--|:--|:--|
| /p/ | /b/ | 爆破音（双唇音） | pen | bed | 双唇闭合后爆破出气 |
| /t/ | /d/ | 爆破音（舌尖音） | ten | dog | 舌尖抵上齿龈爆破出气 |
| /k/ | /g/ | 爆破音（舌后音） | cat | go | 舌后抵软腭爆破出气 |
| /f/ | /v/ | 摩擦音（唇齿音） | fan | van | 上齿咬下唇出气摩擦 |
| /θ/ | /ð/ | 摩擦音（齿音） | thin | this | 舌尖伸出上下齿之间摩擦 |
| /s/ | /z/ | 摩擦音（齿龈音） | see | zoo | 舌尖靠近上齿龈 |
| /ʃ/ | /ʒ/ | 摩擦音（后齿龈音） | she | measure | 舌面靠近硬腭后部 |
| /tʃ/ | /dʒ/ | 破擦音（齿龈音） | church | judge | 先爆破后摩擦，类似“吃/汁”音 |

### 🧠 小贴士
- 手放在喉咙处发音：  
  👉 无震动 → 清辅音  
  👉 有震动 → 浊辅音  
- 记忆顺序建议：  
  1️⃣ 先掌握元音 → 2️⃣ 再练辅音 → 3️⃣ 最后做清浊对比练习。
